# Curso de Python 3 - Do Básico Ao Avançado

Este é, sem sombra de dúvidas, o curso de Python 3 (e versões superiores) mais completo disponível atualmente. O único curso constantemente atualizado para as versões mais recentes do Python gratuitamente.

## Sobre o Curso

São mais de **140 horas de conteúdo**, mais de **745 aulas** sobre assuntos relacionados com a linguagem Python (do básico ao avançado), incluindo várias seções bônus sobre expressões regulares, SQL, Django, HTML e CSS.

## O que Você Aprendendo

| Nível | Conteúdo |
|-------|----------|
| **Iniciante** | Lógica de programação e Python básico |
| **Intermediário** | Programação procedural e funcional (funções, repetições) |
| **Avançado** | Programação orientada a objetos (modo Pythônico) + Design Patterns GoF |

## Tecnologias e Ferramentas Abordadas

- **Python 3.x** - Versões mais recentes da linguagem
- **Django** - Criação de sites e APIs
- **PySide6 (Qt 6)** - Interface Gráfica
- **Selenium** - Automação web
- **SQLite3 / MySQL / MariaDB** - Bases de dados relacionais
- **Pytest / Unittest** - Testes automatizados com TDD
- **Expressões Regulares (Regex)** - Manipulação de padrões textuais
- **HTML5 e CSS3** - Desenvolvimento web front-end
- **SQL com MySQL** - Consultas e manipulação de dados

## Projetos Desenvolvidos

- Gerador e validador de CPF
- Aplicações com Interface Gráfica (PySide6)
- Sites utilizando Django
- Deploy em servidor Linux (Google Cloud Platform)
- Projetos com padrões de projeto GoF
