## 📚 Sobre Este Repositório

Este repositório contém meus estudos e exercícios do curso de Python 3 do básico ao avançado, ministrado pelo professor **Luiz Otávio Miranda**.

O curso aborda desde os conceitos fundamentais até tópicos avançados, incluindo POO, Django, banco de dados e muito mais.
