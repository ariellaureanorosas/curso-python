## 📚 Sobre o Curso

Este é, sem sombra de dúvidas, **o curso de Python 3 mais completo que você encontrará na atualidade**. O único curso constantemente atualizado para as versões mais recentes do Python gratuitamente.

Com mais de **140 horas de conteúdo**, **745+ aulas** e **185.000+ alunos**, este curso cobre tudo o que você precisa para dominar Python — do básico ao avançado, incluindo tecnologias modernas do mercado.

> *"Este curso é dividido em módulos progressivos, permitindo que você aprenda no seu ritmo, do zero até a criação de projetos reais."*

## 🏆 O que Você Vai Aprender

| Nível | Duração | Conteúdo |
|:-----:|:-------:|:--------|
| 🔰 **Iniciante** | ~30h | Lógica de programação, Python básico, variáveis, operadores, estruturas de controle |
| ⚡ **Intermediário** | ~40h | Programação procedural e funcional, funções, repetições, comprehensions, módulos |
| 🚀 **Avançado** | ~50h | Programação orientada a objetos (POO), Design Patterns (GoF), decorators, generators |
| 🎁 **Bônus** | ~20h | Django, PySide6, Selenium, SQL, Regex, HTML5, CSS3, testes automatizados |

## 💻 Projetos Desenvolvidos

Durante o curso, desenvolvemos projetos reais do zero:

- ✅ **Gerador e Validador de CPF** - Validando documentos brasileiros
- ✅ **Sistema de Login** - Autenticação com Django
- ✅ **Blog completo** - Publicação de conteúdo
- ✅ **Dashboard interativo** - Interface com PySide6
- ✅ **Web scraping** - Extração de dados com Selenium
- ✅ **API REST** - Criação de APIs com Django REST
- ✅ **Deploy em produção** - Linux no Google Cloud Platform
- ✅ **Design Patterns** - Implementação de padrões GoF
